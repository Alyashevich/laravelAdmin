## Laravel

В проекте реализована система админстратора, CRUD.

## Зависимости 
PHP 7, MySQL, composer, Laravel 5.3

## Установка 
- cd domains
- клонируем проект git clone https://github.com/Alyashevich/laravelAdmin.git
- cd LaravelAdmin
- composer install
- создать базу Laravel
- .env (прописать подключение)
- php artisan migrate
- php artisan db:seed
